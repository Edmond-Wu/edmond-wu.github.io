Some projects I have worked on!
Web Apps
Project Reports: Web application built using AngularJS with a NodeJS/Express back-end. Users submit project reports and log savings in money and productivity.
Redditer: Ruby-On-Rails chat messaging application that aims to bring together people who subscribe to various Reddit subreddits! Hosted by Amazon Web Services. App is no longer hosted so an Imgur album link is provided instead.
Socket Chat: Another chat application, built using socket.io! Users can log in, view other logged-on users, and change their message colors. Front-end designed with AngularJS and Materialize CSS.
Rutgers Tzu-Ching: Website for the Rutgers chapter of the Tzu-Chi Collegiate Association. Single-page application designed with AngularJS.
Insightweets: HackNY 2015 group web hack using IBM's Bluemix algorithm to find out what people are like based on their tweets (won best IBM hack at HackNY Spring 2015!)
Other Projects
Tower Defense: A basic tower defense game with a geometric/cube theme with some dank memes. Made using Unity 5 with C#; requires this corresponding Unity3D plug-in to work.
Interstellar Shooter: Game resembling a classic space-shooter arcade. Made using Unity 5 with C# scripts. Also needs the Unity3D plugin.
Photo Album: JavaFX CRUD application resembling a photo album. Users can add/edit/delete albums and photos, as well as view/like other users' photos.
Chess: Java project that simulates a complete 2-player chess game with all competitive rules in terminal.
Android Books: Android book database application that allows for users to subscribe to books to receive new chapter updates.
