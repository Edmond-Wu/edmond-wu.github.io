Some projects I have been working on!
Project Reports: Web application built using AngularJS and Foundation CSS with a NodeJS/Express back-end. Users submit project reports and log savings in money and productivity.
Redditer: Chat messaging application that aims to bring together people who subscribe to various Reddit subreddits! Hosted by Amazon Web Services. App is no longer hosted so an Imgur album is provided instead.
Tower Defense: A basic tower defense game with a geometric/cube theme with some dank memes. Made using Unity 5 with C#; requires this corresponding plug-in to work.
Interstellar Shooter: Game resembling a classic space-shooter arcade. Made using Unity 5 with C# scripts. Also needs the Unity plugin.
Rolling Ball: Unity tutorial rolling ball game where users move a ball around to collect objects; requires the Unity plug-in.
Insightweets: HackNY 2015 group hack using IBM's Bluemix algorithm to find out what people are like based on their tweets (won best IBM hack!)
Photo Album: JavaFX CRUD application resembling a photo album. Users can add/edit/delete albums and photos, as well as view/like other users' photos.
Chess: Java project that simulates a complete 2-player chess game with all competitive rules in terminal.
Android Books: Android book database application that allows for users to subscribe to books to receive new chapter updates.
CodeSkulptor Projects: A repository containing a compilation of Python mini-projects built with the CodeSkulptor Python API. Load and run them in CodeSkulptor here.
