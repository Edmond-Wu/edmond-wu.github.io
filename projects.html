Some projects I have been working on!

GITHUB: The-WUUSTER

  • Redditer: Chat messaging application that aims to bring together people who subscribe to various Reddit subreddits! Hosted by Amazon Web Services.
  • Tower Defense: A basic tower defense game with a geometric/cube theme with some dank memes. Made using Unity 5 with C#; requires this corresponding plug-in to work.
  • Insightweets: HackNY 2015 group hack using IBM's Bluemix algorithm to find out what people are like based on their tweets (won best IBM hack!)
  • Song Library: Java GUI project that simulates a song library where a user can add/delete/edit songs.
  • Twitch Plays Hangman: HackRU Spring 2015 hack that uses FFsplit, Twitch and CodeHS API's to host a Hangman game any chat user can play!
  • Interstellar Shooter: Game resembling a classic space-shooter arcade. Made using Unity 5 with C# scripts.
  • Nightmare Survival: Isometric 3-D shooter game where you're in a nightmare and you have to kill zombie toys. Made using Unity 5 and C#.
  • 15 Puzzle: 15 Puzzle Game simulator with a built-in solve function.
